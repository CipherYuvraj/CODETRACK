# CodeTrack - DSA Learning Tracker

A modern DSA (Data Structures & Algorithms) learning tracker with **spaced repetition** system for optimal retention.

## 🚀 Features
- **Spaced Repetition**: 3-7-15 day intervals for optimal learning
- **Progress Tracking**: Track problems by difficulty and topic
- **Dark/Light Themes**: Modern UI with theme toggle
- **Social Integration**: Connect to GitHub and LinkedIn
- **Real-time Data**: Live progress updates and statistics

## 🛠️ Tech Stack
- **Backend**: Node.js + Express.js
- **Storage**: JSON file-based persistence
- **Frontend**: Modern HTML5/CSS3/JavaScript
- **Charts**: Chart.js for visualizations
- **Design**: Responsive with dark/light themes

## 📦 Quick Start
```bash
npm install
npm start
# Open dashboard-modern.html in browser
```
